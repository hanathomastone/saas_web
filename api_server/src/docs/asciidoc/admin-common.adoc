=== Result Code
|===
| Code | 메시지 | 설명

| 200
| API call successful
| Api 호출 성공

| 400
| Bad request (or message)
| 잘못된 요청

| 401
| Authentication failed {Msg}
| 인증 실패

| 403
| Permission error
| 권한 없음

| 405
| Method not allow
| 허용되지 않는 방법

| 416
| Not found Data {DataName}
| 데이터 조회 실패로 인한 처리 불가

| 417
| {FieldName} is required
| 유효성 검증 실패

| 422
| Already data
| 이미 존재하는 데이터

| 500
| Unknown error
| 알수없는 오류(서버 오류)
|===
